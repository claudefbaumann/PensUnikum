# PensUnikum 🏫

> **Pensunikum organisiert Pensen und Schulpläne für ganze Schulverbände — mit Raum für Lehrerwünsche, Lehrplanvorgaben und faire Verteilung.**

---

## Was ist Pensunikum?

Pensunikum ist eine moderne Web-Applikation zur intelligenten Planung und Verwaltung von Lehrpensen innerhalb eines Schulverbands. Das System berücksichtigt:

- 📋 **Lehrplanvorgaben** — Rahmenbedingungen und Pflichtstunden je Stufe
- 🙋 **Lehrerwünsche** — individuelle Präferenzen der Lehrpersonen
- ⚖️ **Faire Verteilung** — automatisierter Abgleich von Wünschen und Anforderungen
- 🏫 **Schulverbands-Übersicht** — zentrale Planung über mehrere Schulhäuser hinweg
